iOS SDK Programming Manual
v2.3.x
Preface
PrinterSDK is a printing SDK specifically developed for the iOS platform, aimed at meeting diverse
printing needs. It supports TSPL, ZPL, CPCL, and ESC command sets, ensuring that you can easily
integrate and control various printers. Whether connected via Bluetooth or Wi-Fi, PrinterSDK provides
stable and efficient connectivity to meet printing requirements in different scenarios. With PrinterSDK,
developers can easily implement complex printing functionalities, thereby enhancing work efficiency and
user experience.
Usage
The PrinterSDK requires a minimum iOS version of 11.0 or later. To integrate the SDK into your project,
follow these steps:
Installation: Add the libPrinterSDK to your project using CocoaPods or by manually including the
framework.
Initialization: Initialize the SDK in your applications delegate or appropriate view controller.
Connecting to Printers: Use the provided methods to discover and connect to available printers via
Bluetooth or Wi-Fi.
Printing Commands: Utilize the supported command sets (TSPL, ZPL, CPCL, ESC) to format and send
print jobs.
Error Handling: Implement error handling to manage connectivity issues and printing errors effectively
Basic Introduction to PrinterSDK Interface Functions
Class Name Description
CPCLCommand Handles CPCL (Comprehensive Printer Command
Language) commands for printer operations.
POSImageTransfer Manages the transfer of image data to POS (Point of
Sale) printers for printing.
TSCBLEManager Facilitates Bluetooth Low Energy (BLE)
communication with TSC printers.
KDS_Log Provides logging functionalities for debugging and
tracking events within the SDK.
POSPrinter Represents a generic POS printer, providing methods
for printing and status management.
TSCCommand Contains common commands and functionalities for
TSC printers.
LabelDocument Manages the structure and content of label documents
to be printed.
POSPrinterSDK The main interface for integrating the PrinterSDK
with POS systems.
TSCPrinterSDK Specific SDK functionalities for TSC printers,
including setup and command execution.
LabelImageTransfer Handles the transfer of label images to printers for
accurate printing.
POSWIFIManager Manages Wi-Fi connections for POS printers, enabling
network-based printing.
TSCWIFIManager Facilitates Wi-Fi communication specifically with
TSC printers.
POSBLEManager Manages Bluetooth connections for POS printers,
ensuring stable communication.
PTable Represents a data structure for handling tabular data,
commonly used in printing contexts.
WIFIConnecter Handles the establishment and management of Wi-Fi
connections for printers.
POSCommand Provides a set of commands for interacting with POS
printers, covering various functionalities.
PrinterProfile Stores configuration and profile information for
different printer models.
ZPLCommand Manages ZPL (Zebra Programming Language)
commands for Zebra printers.
